Device setup

Set up this device

One-time setup so this device can stream to the studio. Takes about a minute.

1

Install the studio certificate

This lets the device trust the local studio over HTTPS. Without it the camera is blocked.

Download certificate
  1. Tap Allow when asked to download a configuration profile.
  2. Open Settings — tap Profile Downloaded near the top, then Install (top-right), and enter your passcode.
  3. Go to Settings ▸ General ▸ About ▸ Certificate Trust Settings and turn ON the "multicam" certificate. (This second step is required — the camera stays blocked without it.)
  1. When the file downloads, open it (or go to Settings ▸ Security ▸ Install a certificate ▸ CA certificate).
  2. Confirm installing it ("Install anyway" if warned).

iPhone: install the downloaded profile, then Settings ▸ General ▸ About ▸ Certificate Trust Settings → enable it.
Android: open the file or Settings ▸ Security ▸ Install a certificate ▸ CA certificate.

2

Open the device page

Once the certificate is trusted, open the device page, enter a name, pick a source, and tap Join.

Open the device page
Point another phone's camera at this code to open this setup page on it.

Why a certificate? The studio runs on your local network with no internet, so it uses a locally‑trusted certificate (mkcert) instead of a public one. You trust it once per device; it keeps working even if the WiFi changes.
